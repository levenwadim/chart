Chart
====

Библиотека для постороения svg-графиков

**Подключение**
```js
let elem = document.querySelector('#chart');
const params = {
   // Массив графиков
   data: [
       // Массив значений графика
       [0, 1, 2, 0],
   ]
};
new Chart(elem, params);
```

**Поддерживаемые параметры**
```js
const params = {
    // Массив графиков
    data: [
        // Массив значений графика
        [0, 1, 2, 0],
    ],

    // Массив подписей для значений графика
    labels: [
      'ПН', 'ВТ', 'СР', 'ЧТ',
    ],

    // Отступы по x и y от краев блока
    padding: 50,

    // Тип графика
    type: Chart.LINE_TYPE,

    // Общие стили для графиков
    style: {
        // Стили для точек
        points: {
            size: 3,
        },
        // Стили для линий
        lines: {
            size: 3,
            bezier: true,
        },
    },
};
```


**Поддерживаемые стили**
```js
points: {
    size: 3,          // Размер точек на графике
    fill: '#222',     // Цвет точек
    stroke: '#fff',   // Цвет обводки точек
    strokeWidth: 3,   // Размер обводки точек
},
lines: {
    size: 3,          // Толщина линий на графике
    fill: '#222',     // Цвет линий
    bezier: true,     // Сглаживание линий по Безье
}
```
